# Admission Forms

This folder should contain the following PDF files:

1. **admission-form.pdf** - School admission application form
2. **requirements.pdf** - List of required documents and items
3. **fees-structure.pdf** - Detailed breakdown of school fees
4. **rules-regulations.pdf** - School rules and regulations

## How to Add PDFs

Simply place your PDF files in this folder with the exact filenames listed above.

The Apply page will automatically link to these files for download.

## Note

Placeholder files can be replaced with actual PDF documents. The download functionality 
is already implemented in the Apply page component.
